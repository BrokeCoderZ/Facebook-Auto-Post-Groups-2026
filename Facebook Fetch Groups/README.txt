[ FB Fetch Group ]

FB Fetch Group is a feature used to automatically retrieve (collect) a list of Facebook groups from your account.
With this feature, you no longer need to search for or add groups one by one to obtain group names or URLs, making the process much faster and more efficient. 
It is ideal for simplifying group management before performing auto posting or other activities. 
Make sure to save the data in JSON format if it will be used for auto posting.

# INFORMATION
Some antivirus programs may detect this application as a false positive. 
This occurs due to the protection system implemented within the application.
If this happens, please add the application to your antivirus whitelist to ensure it runs properly.
